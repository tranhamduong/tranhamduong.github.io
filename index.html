Social Listening In Student Admissions Work
Graduation Thesis

  • Java
  • Apache Spark
  • Deeplearning4j
  • NLP Processing
  • MongoDB
  • JSoup & Selenium Driver

My university graduation thesis, which provides a application for sentiment analysis on social data about university reviews.
Firstly, I used JSoup & Selenium Driver to crawl data from multi internet & social platform sources, then used those data as training data set to a LSTM neuron network (provided by open-source library Deeplearning4j) for analyze sentiment task. The target data which is from top three technology university social activities & interactions are located in a multi-cluster Mongo Database. The whole application is build with Java and Apache Spark Framework for future intention to serve multi cluster processing.

News Website

  • Java

  • Thymeleaf
  • Hibernate
  • MySQL Database
  • JSoup
  • HTML/CSS/Javascript

This project is built firstly for an university subject, but lately has been developed much better for a magazine. The website is built from scratch with Java Spring Boot Framework; organized as a DI and multi-module maven project; MySQL Database and Hibernate Framework to help manage Spring Data JPA repository. It goes with necessary administrator tools and functions to create/ manage posts and website elements, an automatic crawler to assist in generating contents, user-friendly text and upload for editor (using CKEditor, and security protection with Spring Security)

ASP. NET MVC websites

  • C#
  • ASP.NET MVC Framework
  • Entity Framework
  • Razor View Engine
  • MSSQL Server
  • Microsoft Azure
There are two website projects created with ASP.NET MVC Framework, the first one is about a CV Template Provider (the concept is similar to top.vn): client will input their information to the web application, the data then will be stored into database, then be transfered to apply in a various of CV template and be ready to be downloaded as PDF-format file. The second project is about a travel tour booking website, there are both client and administrator activities: the clients can search, view information, compare tours, book tours, read or make reviews, and the admins can create, remove, update tours, check tours info, get list of participants, and receipt of tour bookings. Both projects use similar technology and framework, are builted from scratch, their data is stored in MSSQL Server and is modeled with Entity Framework. The websites and their database are deployed through Microsoft Azure.
